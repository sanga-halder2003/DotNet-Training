# Library Management System

A console-based Library Management System developed using **C#** and **SQL Server** to automate library operations such as managing books, members, authors, categories, and book transactions.

## Features
- Book Management (Add, Update, Delete, View)
- Author & Category Management
- Member Management
- Issue and Return Books
- Book Search
- Reports Generation

## Technologies Used
- C#
- .NET
- SQL Server
- ADO.NET

## Database Tables
- Authors
- Categories
- Books
- Members
- Transactions

## How to Run
1. Create the SQL Server database.
2. Execute the SQL scripts.
3. Update the connection string.
4. Run the C# Console Application.

## Project Objective
To demonstrate database design, normalization, CRUD operations, joins, stored procedures, indexing, and integration of C# with SQL Server.
